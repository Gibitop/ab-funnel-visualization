# AB funner simulations

## First demo

2 step form with 2 variants in each step

### Running

Open the `index.html` file in any browser


## Second demo

Error measuring simulation with configurable steps and variants

### Running

**With node >= v22.6**
```
node --experimental-strip-types second-demo/simulation.ts
```

**With bun**
```
bun second-demo/simulation.ts
```
